# i-Amb Animal

Dashboard estático para consolidar as respostas do bloco de proteção animal do IEGM Ambiental 2024.

## Publicação na Vercel

- Framework preset: `Other`
- Build command: vazio
- Output directory: `.`
- Root directory: raiz do repositório

O painel carrega `data/iamb_animal_2024.csv` diretamente pelo navegador.
